# Adblock_Plus_List
Adblock_Plus_List for small_5 openwrt

# 使用方法一（Win自带SSH可用）

ssh进入路由器

执行

`curl https://gitee.com/fangxx3863/Adblock_Plus_List/raw/master/Adblock_Gitee_mirror.sh|sh`
